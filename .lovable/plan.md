## 3. Taskbar
- Taskbar di bagian bawah layar, bergaya Windows 11 (centered icons)
- Ikon Start Menu, search, dan shortcut aplikasi
- System tray di kanan: toggle dark/light mode, jam & tanggal
- Indikator window yang sedang terbuka

## 4. Start Menu
- Panel Start Menu bergaya Windows 11 saat klik tombol Start
- Menampilkan daftar "aplikasi" (About, Projects, Resume, dll.)
- Pinned apps dan recent items

## 5. Draggable & Resizable Windows
- Setiap "aplikasi" terbuka dalam jendela terpisah
- Jendela bisa di-drag, resize, minimize, maximize, dan close
- Title bar dengan ikon dan nama aplikasi
- Multiple windows bisa terbuka bersamaan dengan z-index management

## 6. Konten Aplikasi
- **About Me**: Foto, bio singkat, social media links
- **Projects**: Grid/list proyek dengan gambar, deskripsi, dan link
- **Resume/CV**: Timeline pengalaman kerja dan pendidikan
- **Skills**: Progress bar atau badge untuk setiap skill
- **Contact**: Form dengan nama, email, dan pesan (frontend only)

## 7. Light/Dark Mode
- Toggle antara Windows 11 light dan dark theme
- Wallpaper berubah sesuai tema
- Semua elemen UI (taskbar, windows, start menu) menyesuaikan
